3    INTRODUÇĂO

3.  INTRODUÇĂO
Este documento foi elaborado, assumindo que o leitor é conhecedor dos conceitos básicos do banco de dados sql server, IIS, browsers, linguagens asp, sql e html. O propósito deste documento é explicar passo-ŕ-passo como deverá ser instalado o melhor e mais completo sistema de suporte para processos R3. Antes de tudo, será dada uma introduçăo básica de banco de dados, IIS, browser e linguagens web e em especial ao sistema X-PROC.

O que é Banco de Dados?
A palavra BANCO DE DADOS representa um arquivo físico de dados, armazenado em dispositivos periféricos, onde estăo armazenados os dados de diversos sistemas, para consulta e atualizaçăo pelo usuário.

O que é IIS ?
A sigla IIS nada mais é que Internet Information Services ou seja, é um modelo de servidor web desenvolvido pela microsoft que permite a criaçăo de diversas aplicaçőes Internet.

O que é Browser ?
Browser é um programa desenvolvido para permitir o acesso  ŕ Internet, funciona como um tradutor de linguagens entre o computador e o homem.

 O que é Linguaguem web ?
Existem vários tipos de linguagens web (asp, sql, java  e etc), mas todas tem por objetivo criar sistemas e páginas (programas) que somente podem ser acessadas através de um browser. 

O que é o Sistema X-PROC ?
X-PROC é um sistema desenvolvido para atender as necessidades de definiçăo do escopo para implementaçăo do SAP R/3 em  todas as unidades de negócios. O X-PROC está desenvolvido em linguagens de programaçőes asp, sql, html e tabelas armazenadas no banco de dados sql server.

Obs.: Faz parte deste documento dois discos laser contendo os softwares e arquivos necessários para a instalaçăo do sistema X-PROC.
